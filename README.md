# MonkeyScripts

A collection of userscripts compatible with browser extensions like [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/) and [Violentmonkey](https://violentmonkey.github.io/).

> **Recommended:** Use [Violentmonkey](https://violentmonkey.github.io/) — it is fully open source ([GitHub](https://github.com/violentmonkey/violentmonkey)), actively maintained, and available for all major browsers.

## Installation

1. Install a userscript manager:
   - [Violentmonkey](https://violentmonkey.github.io/) *(recommended — open source)*
   - [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/) *(Firefox only)*
2. Open the script file you want to install from this repository.
3. Click the raw view of the file — your userscript manager will detect it and prompt you to install.
